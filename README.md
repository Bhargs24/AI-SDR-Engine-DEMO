# AI SDR Engine

AI-powered outbound automation system. Turns ICP definitions into personalized multichannel campaigns.

## Demo Video

> 📹 [Watch 3-minute demo](https://drive.google.com/file/d/10W6tbTBM-0rCoF8ApF4msHBmMJYH3rJN/view?usp=sharing)

## What It Does

- **Input:** Industry, persona, region, lead count
- **Output:** Personalized email + LinkedIn messages in 3 minutes
- **Quality:** 85-95% relevance scoring with AI guardrails

## Results

- **Speed:** 5 leads → 30 messages in 3 min (vs 2.9 hrs manual)
- **Cost:** 98% savings vs traditional SDR work
- **Scale:** Handles 1,000+ leads/month

## Tech Stack

Next.js 14, TypeScript, Prisma, OpenAI GPT-4, Tailwind CSS

## Features

✅ ICP-based lead generation  
✅ AI personalization with quality checks  
✅ Multi-channel orchestration (email + LinkedIn)  
✅ Real-time workflow tracking  
✅ ROI calculator  
✅ Export to CSV

- Agency client prospecting
